What Happens Behind the Scenes
  • Your app is treated like a Canvas App — Even though you wrote React code, the platform classifies it the same as a drag-and-drop app
  • It’s nested inside multiple layers — Like a series of picture frames: platform shell → player → runtime → your code
  • 77 scripts load before yours — For login, security, Teams integration, Copilot AI, user surveys, and more
  • 230KB of translations load every time — Text for every language, device type, and region — most of it will never be used
  • 55 on/off switches — Microsoft can enable or disable features for your organization remotely
What You Get Without Doing Anything
  • Login is handled for you — Users are authenticated before your app even loads — you never touch passwords or tokens
  • Copilot AI is pre-wired — A built-in AI assistant panel is ready to go — 185 references to it in the page code
  • Works in government clouds automatically — Same app code runs in US Gov, Germany, China — no changes needed
  • Accessibility is built in — Screen reader support, keyboard navigation, skip-to-content links, progress indicators
  • Security headers are managed — Content security policies and script protections are applied for you
